# Polylith Installation

## Requirements

In order to build a polylith deployment, you will need:

* Go 1.16 or later
* PostgreSQL 12 or later
* NATS Server 2.8.0 or later

## Build Dendrite

On UNIX systems, the `build.sh` script will build all variants of Dendrite.

```sh
./build.sh
```
